A Module for Apache Version 1.3.x and later


Apache Module mod_limits

Available Languages:  en 

Description: Apache module to protect against DoS using the Range header attack
Status:Extension
Module Identifier:limits_module
Source File:mod_limits.c
Compatibility:Version 1.3.x and later

Summary

mod_limits is aimed at protecting the web server during attacks.
It provides two major functionalities:

  • Limit the maximum number of simultaneous connections
  • Do not serve request if the load is over certain value
This module will not function unless mod_status is loaded and the "ExtendedStatus On" directive is set.


MaxConnsPerIP Directive

Description:Defines the maximum number of connections from a single IP.
Syntax:MaxConnsPerIP number (default: 30)
Context:server config, virtual host
Override:FileInfo
Status:Base
Module:mod_limits
Compatibility:Apache 2.x

This directive sets a limit of the maximum simultaneous connections to the server or to a vhost.

Example

ExtendedStatus On

LoadModule limits_module modules/mod_limits.so

<IfModule mod_limits.c>
   MaxConnsPerIP 30
</IfModule>

MaxLoadAVG Directive

Description:Defines the maximum number of connections from a single IP.
Syntax:MaxLoadAVG number (default: 30)
Context:server config, virtual host
Override:FileInfo
Status:Base
Module:mod_limits
Compatibility:Apache 2.x

This directive sets the maximum load average, above which files will not be served.

Example

ExtendedStatus On

LoadModule limits_module modules/mod_limits.so

<IfModule mod_limits.c>
   MaxLoadAVG 10
</IfModule>
